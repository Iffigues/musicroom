**MUSICROOM**

free loving radio station for phone

**SEVER**

----------------------------

**CONFIGURATION**

file configuration: ./conf/ini.ini

-------------------------------

**START**: 

./musicroom (args)

---------------------------- 

**ARGS**:

  
  **no-args**: start server in main thread
  
  **daemon**: start daemon in daemon mod
  
  **reset**:  erase all bdd of musicroom


--------------------------

**kill daemon**
  

./server/kill.sh

--------------------------
